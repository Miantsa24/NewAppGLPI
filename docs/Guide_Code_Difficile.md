# Guide — Comprendre et reproduire les parties "difficiles" du code

Ce guide a été **découpé en plusieurs fichiers**, un par sujet, pour être plus
facile à réviser. Chaque fichier suit la même structure : **Façon de faire**
(la recette générale), **Extrait réel** (un morceau de code du projet),
**Fonctionnement détaillé, fonction par fonction** (ce que reçoit/fait/renvoie
chaque fonction), et **À retenir** (ce qu'il faut savoir pour l'écrire à la
main sans copier-coller).

Aller au nouveau guide : [docs/guide/README.md](guide/README.md)
